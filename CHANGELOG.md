# Changelog

## [1.0.0] - 2025-03-25

Initial release.

- Display total numeric armor value above the HUD armor bar
- Show `30 (Actual: X)` when armor exceeds the vanilla cap of 30
- Read server-synced armor attributes (bonus armor from plugins or gear systems)
- Dynamically adjust position for multi-row health bars and absorption hearts
- Configurable text color (White / Yellow / Cyan / Green)
- Toggle display on/off in-game
- Rebindable config menu key (default: `N`)
- English and Chinese Simplified (简体中文) localization
